# Three Body Problem - Blender Addon

Blender 上で複雑な三体問題（Three-Body Problem）の数値シミュレーションを行い、天体のアニメーションを自動生成するアドオンです。

## 特徴

- **厳選されたプリセットモデル**: 制御の難しい三体問題において、数学的に興味深い挙動を示す6つのモデルをプリセットとして用意。
- **高精度な数値計算**: 4次ルンゲ＝クッタ法（RK4）を採用し、サブステップ計算により安定した軌道を再現。
- **3Dシミュレーション**: 平面的な動きだけでなく、3次元的に複雑に絡み合う立体的な軌道解（Dragon, Yarn 等）にも対応。
- **自動ベイク機能**: シミュレーション結果を直接 Blender のキーフレームとして書き出し、即座にレンダリングや編集が可能。
- **スケーラブルな調整**: 物理的な整合性を保ったまま、ビューポート上のサイズ（Scale）や速度（Time Scale）を自由に調整可能。

## 搭載プリセット

### 2D / 準安定モデル
- **Figure-Eight**: 3つの天体が「8の字」を描きながら追いかけ合う最も有名な安定解。
- **Lagrangian Points (L4)**: 重い二天体のラグランジュ点において、小さな天体が安定して留まる系。
- **Pythagorean**: 3:4:5 の質量比を持つ三体が、カオス的で激しい相互作用の末に一星が弾き飛ばされる古典的問題。

### 3D / 立体モデル
- **The Dragon**: 空間をひねりながら移動する、3次元的な8の字の変種。
- **Butterfly 3D**: 二天体のペアの周囲を、第三の天体が「蝶の羽」のように上下左右に大きく横断。
- **The Yarn**: 特定の平面に捕らわれず、空間を毛糸玉のように均等に埋めていく高難度な3D解。

## 使用方法

1. **セットアップ**: 3つの天体用オブジェクト（UV球など）をシーンに作成します。
2. **オブジェクト指定**: サイドバーの `three_body_problem` タブを開き、Body 1〜3 に作成したオブジェクトを割り当てます。
3. **モデル選択**: `Preset Model` から好みの挙動を選択します。
4. **調整**: `Simulation Scale` で広がりを、`Time Scale` で速さを調整します。
5. **実行**: `Bake Simulation` ボタンを押すと、指定したフレーム範囲にキーフレームが生成されます。

## Tips
- **軌跡の表示**: Blender の `Motion Paths`（オブジェクトプロパティ > モーションパス）を有効にすると、美しい幾何学的な軌道を確認しやすくなります。
- **開発時のデバッグ**: VS Code の `Blender Development` 拡張機能を用い、アドオンのロード/リロードが可能です。
